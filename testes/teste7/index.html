Testando modelos de caixa

Como são formadas as caixas?

  • Heigth, significa altura;
  • Width, significa largura;
  • Border, significa borda;
  • Padding, significa preenchimento interno;
  • Margin, significa margem externo;
  • Header, significa cabeçalho
  • nav, significa navegação
  • main, significa o conteúdo
  • article, significa artigo
  • footer, significa Ródape
